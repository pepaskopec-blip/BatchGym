# 🏋️ BG Gym Neratovice Simulator

Welcome to my **very first game**! I created this project as a **freshman in high school** to learn the basics of programming logic and variables.

It is a text-based RPG built entirely in a Windows Batch script that simulates a real workout session at the local gym in Neratovice.

---

### 📋 Project Overview
* **Origin**: My first-ever programming project (High School Freshman year).
* **Technology**: Windows Batch Script.
* **Goal**: Complete your workout with maximum money and zero injuries.

---

### ⚡ Key Features
* **Experience Selection**: Tailored advice for Beginners, Intermediates, or Advanced gym-goers.
* **Interactive Exercises**: Train Bench Press, Incline Bench, and Triceps.
* **ASCII Animations**: Visual barbell movement rendered directly in the console.
* **Gym Economy**: Start with 1,500 CZK. Manage your budget for entry fees and supplements.
* **Risk System**: Skipping warm-ups or overdoing cardio leads to the hospital.

---

### 🚀 How to Run
1. Download the **.bat** file.
2. Open it on any Windows machine.
3. Enjoy the grind!

**❗ IMPORTANT:**
Don't forget to stretch! If you go straight to heavy weights without a warm-up, it's game over in the hospital.
